conf
====

Linux base config


## Local config

Anything private should be in `~/.bash_aliases_local`

## Necessary binaries

- rg
- zathura
- iwctl / wget
- sway / waybar / mako

## qutebrowser and rosepine

```sh
git clone https://github.com/aalbegr/qutebrowser-rose-pine.git ~/.config/qutebrowser/rosepine
```
